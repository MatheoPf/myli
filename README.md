# Utilisation de l’IA dans le projet

Nous avons utilisés Mistral AI dans ce projet afin d’améliorer la qualité du code et de faciliter certaines tâches non techniques.

### Regroupement et traduction des genres musicaux

- Regroupper des sous-genres de musiques en catégories plus globales.
- Traduire certains styles en anglais

### Nettoyage et amélioration du code

- Réécrire certains blocs de code pour les rendre plus clairs et plus lisibles.
- Proposer des améliorations structurelles ou stylistiques.
- Optimiser des passages répétitifs ou peu maintenables.

L’objectif n’était pas de déléguer la logique métier, mais d’utiliser l’IA comme un assistant pour améliorer la qualité du code.

---

En résumé, l’IA n’a pas remplacé le processus de développement, mais a joué un rôle d’outil complémentaire pour organiser les données musicales et améliorer le code.
